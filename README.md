MILA – iOS STORYTELLING APPLICATION

MILA – Interactive Storytelling iOS Application
An iOS app built using Swift for exploring and managing interactive stories.

What It Does:

Displays interactive stories categorized by type.
Allows users to save favorite stories.
Maintains structured story data using Swift models.
Provides smooth and intuitive user interface.

REQUIREMENTS :

Swift
Xcode
iOS Simulator / Device

How this App works :

User browses stories based on categories.
User can select and read stories.
Favorites feature allows saving preferred stories.
App uses structured data models for efficient handling.

Purpose :

This application provides an engaging platform for users to explore and manage storytelling content.

Compilation Steps :

Install Xcode.
Open the project.
Run the application on simulator or device.
